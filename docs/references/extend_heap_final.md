# `extend_heap()` — 힙 확장 함수 (테스트케이스 포함)

---

## 함수 코드

```c
static void *extend_heap(size_t words) {
    char *bp;
    size_t size;

    size = (words % 2) ? (words + 1) * WSIZE : words * WSIZE;

    if ((bp = mem_sbrk(size)) == (void*)-1) {
        return NULL;
    }

    PUT(HDRP(bp), PACK(size, 0));
    PUT(FTRP(bp), PACK(size, 0));
    PUT(HDRP(NEXT_BLKP(bp)), PACK(0, 1));

    return coalesce(bp);
}
```

---

## 테스트케이스 설정

```
mm_malloc(13) 호출
  → asize = ALIGN(13 + 8) = 24B
  → find_fit 실패
  → extendsize = MAX(24, 4096) = 4096B
  → extend_heap(4096 / 4 = 1024 words) 호출

힙 상태 (호출 직전):
  이전 블록:     0x2004, 크기=32B, free
  에필로그 헤더: 0x2024, PACK(0,1)
  mem_brk:      0x2028
```

---

## Step 1 — 더블워드 정렬

```c
size = (words % 2) ? (words + 1) * WSIZE : words * WSIZE;
```

```
words = 1024

1024 % 2 == 0  →  짝수
size = 1024 × 4 = 4096B  ✓
```

### 왜 짝수여야 하는가?

```
블록은 8바이트(더블워드) 배수여야 함

words = 3  →  3 × 4 = 12B  (12 % 8 ≠ 0 → 정렬 위반!)
words = 4  →  4 × 4 = 16B  (16 % 8 == 0 → 정렬 OK  ✓)

홀수 보정 예시:
words = 1023  →  (1023+1) × 4 = 4096B  ✓
words = 7     →  (7+1)    × 4 =   32B  ✓
```

---

## Step 2 — `mem_sbrk(4096)` : 힙 확장

```c
if ((bp = mem_sbrk(size)) == (void*)-1)
    return NULL;
```

### 확장 전 힙

```
주소       내용
────────────────────────────────────────────────────────
0x2000     이전 블록 헤더   PACK(32, 0)  ← free
0x2004     이전 블록 페이로드 시작
...
0x2020     이전 블록 풋터   PACK(32, 0)
────────────────────────────────────────────────────────
0x2024     에필로그 헤더    PACK(0, 1)   ← mem_brk = 0x2028
────────────────────────────────────────────────────────
```

```
mem_brk 기준:

┌─────── ... ───┬──────────┐
│   기존 블록들  │  에필H   │
└─────── ... ───┴──────────┘
                            ↑ mem_brk = 0x2028
```

### `mem_sbrk(4096)` 호출 후

```
old_brk = 0x2028  (기존 mem_brk 저장)
mem_brk += 4096   → mem_brk = 0x3028
반환값 = old_brk = 0x2028
```

```
┌─────── ... ───┬──────────┬──────────────────────────────┐
│   기존 블록들  │  에필H   │       새로 확보된 4096B      │
└─────── ... ───┴──────────┴──────────────────────────────┘
                ↑ bp = 0x2028              ↑ 새 mem_brk = 0x3028
```

### 핵심 — `bp`가 가리키는 위치

```
bp = old_brk = 0x2028

HDRP(bp) = bp - WSIZE(4) = 0x2028 - 4 = 0x2024
                                           ↑ 기존 에필로그 헤더 자리!

→ 기존 에필로그가 새 블록의 헤더 자리로 재활용됨
```

## Step 3 — `PUT` × 3 : 가용 블록 구조 기록

```c
PUT(HDRP(bp),            PACK(size, 0));  // 새 가용 블록 헤더
PUT(FTRP(bp),            PACK(size, 0));  // 새 가용 블록 풋터
PUT(HDRP(NEXT_BLKP(bp)), PACK(0, 1));    // 새 에필로그 헤더
```

### 매크로 주소 계산 (bp=0x2028, size=4096)

```
HDRP(bp)        = bp - 4           = 0x2028 - 4    = 0x2024
FTRP(bp)        = bp + size - 8    = 0x2028 + 4088 = 0x302C
NEXT_BLKP(bp)   = bp + size        = 0x2028 + 4096 = 0x3028
HDRP(NEXT_BLKP) = bp + size - 4    = 0x2028 + 4092 = 0x3030
```

### 기록 전

```
주소      내용
────────────────────────────────────────────────────────────
0x2024    구 에필로그 헤더  PACK(0,1)    ← HDRP(bp) = 0x2024
0x2028    날것의 메모리 시작              ← bp
...
0x302C    ??                             ← FTRP(bp) = 0x302C
0x3030    ??                             ← 새 에필로그 자리
────────────────────────────────────────────────────────────
```

### 기록 후

```
주소      내용
────────────────────────────────────────────────────────────
0x2024    새 블록 헤더    PACK(4096, 0)  alloc=0  ← HDRP(bp)
0x2028    페이로드 시작                           ← bp
...
0x302C    새 블록 풋터    PACK(4096, 0)  alloc=0  ← FTRP(bp)
0x3030    새 에필로그H    PACK(0, 1)    alloc=1
────────────────────────────────────────────────────────────
```

```
전체 블록 레이아웃:

 0x2024    0x2028                    0x302C     0x3030
┌─────────┬──────────────────────────┬─────────┬────────┐
│ 헤더    │     페이로드 공간        │ 풋터    │새 에필H│
│PACK     │        (free)            │PACK     │PACK    │
│(4096,0) │                          │(4096,0) │(0,1)   │
│alloc=0  │                          │alloc=0  │alloc=1 │
└─────────┴──────────────────────────┴─────────┴────────┘
↑ HDRP(bp) ↑ bp=0x2028              ↑ FTRP(bp) ↑ 새 에필로그
```

## Step 4 — `coalesce(bp)` : 인접 블록 병합

```c
return coalesce(bp);
```

### 이전 블록 상태 확인

```
PREV_BLKP(bp)    = bp - GET_SIZE(bp - DSIZE)
                 = 0x2028 - GET_SIZE(0x2020)
                 = 0x2028 - 32
                 = 0x2004         ← 이전 블록 페이로드

GET_ALLOC(FTRP(PREV_BLKP(bp))) = GET_ALLOC(0x2020) = 0  →  free!
```

→ 이전 블록이 free이므로 **Case 3** (앞과 병합) 발생

### Case 3 병합 과정

```
병합 전:
 0x2000      0x2004               0x2020 0x2024 0x2028
┌──────────┬──────────────────────┬──────┬──────┬──────────────────────┬──────┐
│ 이전 헤더 │    이전 페이로드     │이전F │현재H │   현재 페이로드       │새에필│
│PACK(32,0)│       (free)         │PACK  │PACK  │      (free)          │PACK  │
│ alloc=0  │                      │(32,0)│(4096,0)│                   │(0,1) │
└──────────┴──────────────────────┴──────┴──────┴──────────────────────┴──────┘
              ↑ PREV_BLKP(bp)=0x2004              ↑ bp=0x2028

size = 32 + 4096 = 4128B
bp   = PREV_BLKP(bp) = 0x2004

PUT(FTRP(bp),            PACK(4128, 0))  → 0x302C에 갱신
PUT(HDRP(PREV_BLKP(bp)), PACK(4128, 0)) → 0x2000에 갱신

병합 후:
 0x2000      0x2004                                        0x302C 0x3030
┌──────────┬──────────────────────────────────────────────┬──────┬──────┐
│ 병합 헤더 │              병합된 페이로드 공간            │ 풋터 │에필H │
│PACK(4128,0)│                  (free)                    │PACK  │PACK  │
│ alloc=0  │                                              │(4128,0)│(0,1)│
└──────────┴──────────────────────────────────────────────┴──────┴──────┘
  ↑ HDRP                                                          ↑ 새 에필로그
  ↑ bp = 0x2004 반환
```

## 최종 반환 및 `mm_malloc` 연결

```
extend_heap이 bp=0x2004 반환
  ↓
mm_malloc에서:
  place(bp=0x2004, asize=24)

병합된 블록 csize = 4128B
csize - asize = 4128 - 24 = 4104 ≥ 16 → 분할 발생

분할 후:
 0x2000      0x2004          0x201C  0x2020  0x2024
┌──────────┬──────────────────┬──────┬──────┬────────────── ...
│ 헤더     │    페이로드      │ 풋터 │새가용│   가용 블록(4104B)
│PACK(24,1)│    (24B 할당)    │PACK  │ 헤더 │
│ alloc=1  │                  │(24,1)│PACK  │
└──────────┴──────────────────┴──────┴──────┴────────────── ...
             ↑ 사용자에게 반환되는 bp=0x2004
```

## 전체 흐름 요약

```
extend_heap(1024 words)
  │
  ├─ Step 1: 1024 % 2 == 0  →  size = 4096B
  │
  ├─ Step 2: mem_sbrk(4096)
  │    bp = old_brk = 0x2028
  │    HDRP(bp) = 0x2024  ← 기존 에필로그 재활용
  │
  ├─ Step 3: PUT × 3
  │    0x2024 ← PACK(4096, 0)  헤더
  │    0x302C ← PACK(4096, 0)  풋터
  │    0x3030 ← PACK(0, 1)     새 에필로그
  │
  └─ Step 4: coalesce(0x2028)
       이전 블록(0x2004, 32B) alloc=0 → Case 3 병합
       size = 32 + 4096 = 4128B
       bp = 0x2004 반환
            ↓
       mm_malloc → place(0x2004, 24) → 분할 → 사용자에게 반환
```
